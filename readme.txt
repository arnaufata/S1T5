S1T5 - Instruccions per Executar els Exercicis

INTRODUCCIÓ

Aquest projecte conté 5 exercicis, cadascun dins del seu propi paquet. A continuació s'expliquen els passos per executar cada 
exercici des de la línia de comandes.

ACCEDIR AL TERMINAL
Des de Windows

1. Premeu 'Win + R'.
2. Escriviu 'cmd' i premeu 'Enter'.
3. Navegueu fins al directori del projecte amb la següent comanda: 'cd C:\Users\arnal\IdeaProjects\S1T5\src\main\java'

Des del terminal d'IntelliJ:

1. Obriu el terminal integrat d'IntelliJ IDEA.
2. Navegueu fins al directori 'java' del projecte.

EXERCICI 1:

Compilar: javac n1exercici1/*.java
Executar: java -cp . n1exercici1.Main "C:\Users\arnal\IdeaProjects\S1T5\Directori"

EXERCICI 2:

Compilar: javac n1exercici2/*.java
Executar: java -cp . n1exercici2.Main "C:\Users\arnal\IdeaProjects\S1T5\Directori"

EXERCICI 3:

Compilar: javac n1exercici3/*.java
Executar: java -cp . n1exercici3.Main "C:\Users\arnal\IdeaProjects\S1T5\DirectoriAmbSubs"


EXERCICI 4:

Compilar: javac n1exercici4/*.java
Executar: java -cp . n1exercici4.Main "C:\Users\arnal\IdeaProjects\S1T5\src\main\java\n1exercici4\LlistaCompra.txt"


EXERCICI 5:

Compilar: javac n1exercici5/*.java
Executar: java -cp . n1exercici5.Main "C:\Users\arnal\IdeaProjects\S1T5\objecte_serialitzat.ser"


ALTRES ACCIONS QUE ENS PODEN SER ÚTILS PER MOURE'NS PEL TERMINAL:

1. dir          --> veiem tot el qe hi ha a la carpeta
2. cd (carpeta) --> per anar a la carpeta desitjada
3. cd..         --> per tornar anrere
4. mkdri        --> crea una carpeta on estàs (un cop creada no entra directament, has de posar cd (nomCarpeta))
